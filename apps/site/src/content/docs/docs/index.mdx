ContextCake keeps company rules, team practices, and personal notes in separate layers. AI tools receive one resolved answer without losing the source of each section or any disagreement.

## What to expect

ContextCake is an open source engine. It resolves layers when a tool asks for context, attaches the source to each part of the answer, and reports conflicting values instead of blending them together.

[Packs](/packs) are sold separately. Each Pack contains files for a specific workflow and installs as a base layer, so your team and personal changes can stay separate.
