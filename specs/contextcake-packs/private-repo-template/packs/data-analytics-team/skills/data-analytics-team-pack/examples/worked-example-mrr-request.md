# Worked Example: MRR Request {#worked-example-mrr-request}

Walks a single request — "what's our MRR trend?" from a Northwind sales
lead — through all four phases of the hero workflow, start to finish.

## What it shows {#what-it-shows}

The original vague ask, the clarifying questions and resulting spec, the
build step, the validation pass against the source of truth, and the
final insight summary — one continuous thread instead of four disconnected
docs.

## Why a worked example {#why-it-matters}

The workflows and templates describe the shape of the process; this shows
it actually running, including the kind of ambiguity that shows up in a
real MRR request and how the pack resolves it.

_The complete version ships with the pack._
