# Source of Truth {#source-of-truth}

Defines what "source of truth" means for a Northwind-style analytics team:
which system wins when the warehouse, the CRM, and a spreadsheet disagree
on the same number.

## Why it's a glossary entry, not a rule {#why-it-matters}

Teams argue about numbers because they haven't agreed on vocabulary. This
entry pins down terms like canonical metric, system of record, and derived
figure so an agent — and a new hire — can tell which value to trust
without re-litigating it on every request.

## What the full entry covers {#what-it-covers}

The authored version lists the team's actual precedence order across
common systems, defines edge cases (backfilled data, mid-quarter
revisions), and cross-links to the policy that operationalizes it.
