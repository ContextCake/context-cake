# Using with Claude Code {#using-with-claude-code}

Practical notes on loading this pack into Claude Code so the hero workflow
actually triggers when a stakeholder request shows up, instead of sitting
unused as reference material.

## What it covers {#what-it-covers}

Where the pack's skill fits relative to a project's own CLAUDE.md, how to
confirm the workflow is being followed rather than skipped under time
pressure, and how to point the agent at the team's real data sources so
the Validate phase has something to reconcile against.

## Why a tool-specific guide {#why-it-matters}

A pack that only describes process in the abstract doesn't change what
happens in a real session. This guide is the bridge from "the team agreed
on this process" to "the agent actually runs it."

_The complete version ships with the pack._
