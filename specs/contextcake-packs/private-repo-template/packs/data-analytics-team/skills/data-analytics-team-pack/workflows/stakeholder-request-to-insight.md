# Stakeholder Request to Insight {#stakeholder-request-to-insight}

The hero workflow of the pack: the end-to-end path a request takes from a
vague Slack message to a delivered insight, in four phases.

## The four phases {#the-four-phases}

Clarify turns an ambiguous ask into a written spec. Build produces the
analysis against that spec. Validate reconciles the result against the
team's source of truth before anyone sees it. Deliver packages the answer
with confidence level, caveats, and a recommended next action — not just
a number.

## Why it's the hero workflow {#why-it-matters}

Every other workflow, template, and prompt guide in this pack is a
sub-routine of this one. Once an agent internalizes this shape, it stops
treating analytics requests as one-shot Q&A and starts treating them as a
process with checkpoints.

_The complete version ships with the pack._
